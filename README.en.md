# 汽车行人检测系统

#### Description
通过交互式的按钮选择，可以选择显示特定类别的检测结果，同时在图像上显示每个类别的计数信息。当点击运行时，程序会将trafic-data数据集里的照片转换成视频呈现在窗口中。同时，视频中的每一帧照片都被加上本人的姓名和学号作为水印，以及三个按钮：“all”，“person”，“car”。分别代表着检测不同的目标类别。一共有22种类别。
