# 汽车行人检测系统

#### 介绍
通过交互式的按钮选择，可以选择显示特定类别的检测结果，同时在图像上显示每个类别的计数信息。当点击运行时，程序会将trafic-data数据集里的照片转换成视频呈现在窗口中。同时，视频中的每一帧照片都被加上本人的姓名和学号作为水印，以及三个按钮：“all”，“person”，“car”。分别代表着检测不同的目标类别。一共有22种类别。

#### 使用说明
本文主要通过交互式的按钮选择，可以选择显示特定类别的检测结果，同时在图像上显示每个类别的计数信息。当点击运行时，程序会将trafic-data数据集里的照片转换成视频呈现在窗口中。同时，视频中的每一帧照片都被加上本人的姓名和学号作为水印，以及三个按钮：“all”，“person”，“car”。分别代表着检测不同的目标类别。

当用户使用鼠标左键点击在按钮区域中，程序就会识别出用户想要进行什么目标检测。并且在视频的左上角区域输出检测出目标对象的数量。

当用户点击在“all”按钮区域中，该程序就会识别出要检测所有目标，这时程序会将所有目标都加上矩形框同时计算出所有目标对象的数量，以及对应的预测目标对象和置信度。并呈现在视频中。                
当用户点击在 “person”按钮区域中，该程序就会识别出要检测行人，这时程序会将视频中所有目标对象为行人的目标加上矩形框同时计算出目标对象为行人的数量，以及对应的预测目标对象“person”和置信度。并呈现在视频中。
当用户点击在“car”按钮区域中，也是相同的道理。
